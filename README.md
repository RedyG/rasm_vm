# RedyLang VM
A bytecode interpreter written in C.

## Main features
* stack machine bytecode
* module/import system
* basic arithmetic
* local variables
* memory allocation on stack and heap ( malloc and alloca )

## Exemple ( fibonacci of 10 )

```asm
fn 0 (args: 0, locals: 0): // main
  i8_const 10
  call 1
  exit

fn 1 (args: 1, locals: 0): // fib
  local_get 0
  i8_const 2
  i32_lt
  br_false 0
_:
  local_get 0
  ret
0:
  local_get 0
  i8_const 1
  i32_sub
  call 1
  local_get 0
  i8_const 2
  i32_sub
  call 1
  i32_add
  ret
```
